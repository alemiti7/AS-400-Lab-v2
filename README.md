# AS/400 Lab — Laboratório de Comandos Técnicos

Laboratório prático de comandos básicos e intermediários para manutenção e operação técnica do **IBM AS/400 / iSeries / IBM i**.

## 🌐 GitHub Pages

Acesse a interface interativa em: [https://alemiti7.github.io/AS-400-Lab-v2/](https://alemiti7.github.io/AS-400-Lab-v2/)

## 📁 Estrutura

Todos os arquivos estão no diretório raiz (sem subdiretórios):

| Arquivo | Conteúdo |
|---------|----------|
| `index.html` | Página principal com busca de comandos |
| `mod-01-introducao.html` ... `mod-11-redes.html` | 11 módulos de documentação |
| `lab-01-basico.html` ... `lab-05-manutencao.html` | 5 laboratórios práticos |
| `script-monitoramento.html` | Script CLP de monitoramento |
| `script-limpeza-spool.html` | Script CLP de limpeza de spool |
| `script-backup-biblioteca.html` | Script CLP de backup de biblioteca |
| `referencia-rapida.html` | Card de referência rápida de comandos |

## 🚀 Publicar no GitHub Pages

1. Crie o repositório no GitHub
2. Faça push de todos os arquivos HTML para `main`
3. Em **Settings → Pages**, selecione branch `main`, pasta `/` (root)
4. Acesse: `https://seu-usuario.github.io/as400-lab/`
